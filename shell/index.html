WebAssembly (Wasm) is an assembly-like low-level language that runs on the web. It is a compilation target for langauges like Rust, AssemblyScript (Typescript-like), Emscripten (C/C++), and much more! Wasm offer a compact binary format, with predictable performance to run alongside Javascript. Wasm is currently shipped in all major browsers, and has runtimes meant for running on servers or interfacing with systems using WASI. To get started, browse the full list of examples below.
